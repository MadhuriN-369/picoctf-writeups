## Problem Overview
The challenge has provided us with a file and we must extract the flag from it.

---

## Key Concept
We need to use basic Linux commands to retrieve the flag.
1) To download a file: Copy the link address (URL) and use wget command to download the file.
   ```bash
   wget URL
   ```
2)To read contents of a file: Use cat command to see the contents of the file
  ```bash
  cat filename.txt
  ```

---

## Solution Walkthrough
- Downloaded the provided file using `wget` command
- Used the `cat` command to display its contents
- The flag was directly visible in the output
